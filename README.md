#  ClimateData

## Описание
ClimateData — это Android-приложение для анализа и учета климатических данных, предназначенное для садоводов и фермеров.
Приложение демонстрирует обработку больших массивов климатической информации, их агрегацию и анализ с целью оценки условий выращивания сельскохозяйственных культур.

## Возможности
- Сбор и хранение климатических данных (температура, влажность, почвенные показатели)
- Генерация тестовых климатических данных большого объема для анализа
- Анализ климатических условий и сопоставление их с требованиями сельскохозяйственных культур
- Учет оптимальных параметров выращивания растений (температура, влажность, сроки роста)
- Удобный интерфейс, адаптированный под мобильные устройства на Android

## Технологии
- **Язык программирования:** Kotlin
- **Платформа:** Android SDK
- **Компоненты Android:** Activity, Fragment, ViewModel, Layout, Intent
- **Внешний API:** OpenWeatherMap

## Установка и запуск
- Клонировать репозиторий:
git clone https://github.com/IamTomatik/ClimateData.git
- Открыть проект в Android Studio
- Дождаться загрузки зависимостей и завершения сборки Gradle
- Запустить приложение на эмуляторе или физическом Android-устройстве, нажав Run

## Структура проекта
- app/src/main/java/... — исходный код приложения (экраны, логика, модели климатических и аграрных данных)
- app/src/main/res — ресурсы приложения (разметка экранов, стили, строки, иконки)
- AndroidManifest.xml — основные настройки приложения и точка входа
